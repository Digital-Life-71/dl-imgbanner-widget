Виджет позволяет быстро и просто доавить баннер картинку на сайт

== Description ==

Просто, удобно, быстро. Достаточно нажать кнопку 'загрузить/выбрать картинку из медиафайлов' указываем заголовок и сссылку, все готово.

== Installation ==

1. Загружаем плагин в директорию `/wp-content/plugins/` 
2. Активируем DL-IMGBanner widget в меню WordPress раздел 'Плагины'
